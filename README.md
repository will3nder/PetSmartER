# PetSmartER

A project for ECEN 1400

## Components:
* Rasberry Pi PICO 2W
* High Torque Servos
* XIAO RA4M1
* 2x Adafruit RFM95W LoRa Radio
* Adafruit Ultimate GPS Breakout - PA1616S  

## Modules:
* A dog collar with GPS trakcing and LoRa communication
* An automatic doggie door
